## 📋 O projekcie

**Prostynet** to inicjatywa edukacyjna dedykowana osobom, które chcą zacząć korzystać z internetu, smartfonów i komputerów. Projekt zawiera przystępne tutoriale, poradniki i zasoby edukacyjne stworzone z myślą o absolutnych początkujących.

Nasze motto: *Technologia powinna być dostępna dla każdego* ✨

## 🎯 Cele projektu

- ✅ Ułatwić dostęp do edukacji technologicznej
- ✅ Zbudować zaufanie wśród osób lękowych przed technologią
- ✅ Dostarczyć praktyczne, krok-po-kroku poradniki
- ✅ Stworzyć wspierającą społeczność uczących się

## 🚀 Szybki Start

1. **Wybierz temat** - Przejdź do folderu `/tutorials`
2. **Przeczytaj poradnik** - Wszystkie pliki w formacie Markdown (łatwo czytelne)
3. **Przećwicz** - Każdy poradnik zawiera praktyczne ćwiczenia
4. **Pytania?** - Otwórz [Issue](https://github.com/tomaszhallek7-dotcom/Prostynet/issues) lub sprawdź [FAQ](./resources/faq.md)

## 📖 Jak korzystać z tego repozytorium?

- 📱 **Na telefonie**: Kliknij na plik, aby go przeczytać
- 💻 **Na komputerze**: Przejrzyj foldery i wybierz temat
- 🔍 **Szukanie**: Użyj funkcji wyszukiwania (Ctrl+F lub Cmd+F)
